# Decoda v1.0.0 BETA #

A lightweight textarea editor with toolbar functionality for the Decoda markup language.

## Requirements ##

MooTools 1.4
* Core
* More/Element.Forms
* More/Element.Shortcuts

Browsers
* Chrome 9+
* Firefox 4+
* Safari 5.1+
* Opera 11+
* Internet Explorer 8+
